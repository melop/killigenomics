Bioclim correlations.

1. Download bioclim variables (*.bil files) from http://biogeo.ucdavis.edu/data/climate/worldclim/1_4/grid/cur/bio_2-5m_bil.zip
2. Extract bioclim variables given GPS coordinates using getBioFactors.R.
3. Edit and run pic.R to perform model selection and correlation using PGLS.
