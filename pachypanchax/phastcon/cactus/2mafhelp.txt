usage: hal2mafMP.py [-h] [--numProc NUMPROC] [--sliceSize SLICESIZE]
                    [--splitBySequence] [--smallSize SMALLSIZE]
                    [--cacheMDC CACHEMDC] [--cacheRDC CACHERDC]
                    [--cacheBytes CACHEBYTES] [--cacheW0 CACHEW0] [--inMemory]
                    [--refGenome REFGENOME] [--refSequence REFSEQUENCE]
                    [--refTargets REFTARGETS] [--start START]
                    [--length LENGTH] [--rootGenome ROOTGENOME]
                    [--targetGenomes TARGETGENOMES] [--maxRefGap MAXREFGAP]
                    [--noDupes] [--noAncestors] [--onlySequenceNames]
                    halFile mafFile

Multi-Process wrapper for hal2maf.

positional arguments:
  halFile               Input HAL file
  mafFile               Output MAF file

optional arguments:
  -h, --help            show this help message and exit
  --numProc NUMPROC     Maximum number of processes to create. If neither
                        --sliceSize or --splitBySequence are specified, then
                        the reference genome will be sliced to require
                        --numProc jobs (default: 1)
  --sliceSize SLICESIZE
                        Maximum slice of reference sequence to process in a
                        single process. (default: None)
  --splitBySequence     Create an output MAF for each sequence in the
                        reference genome. Output files will have the format
                        mafFile_sequenceName.maf (default: False)
  --smallSize SMALLSIZE
                        If --splitBySequence is used, then all sequences with
                        length less than smallSize will be lumped into a
                        single output MAF called "mafFile_small.maf" (default:
                        0)

HDF5 HAL Options:
  --cacheMDC CACHEMDC   number of metadata slots in hdf5 cache (default: None)
  --cacheRDC CACHERDC   number of regular slots in hdf5 cache. should be a
                        prime number ~= 10 * DefaultCacheRDCBytes / chunk
                        (default: None)
  --cacheBytes CACHEBYTES
                        maximum size in bytes of regular hdf5 cache (default:
                        None)
  --cacheW0 CACHEW0     w0 parameter fro hdf5 cache (default: None)
  --inMemory            load all data in memory (& disable hdf5 cache)
                        (default: False)

hal2maf Options:
  --refGenome REFGENOME
                        name of reference genome (root if empty) (default:
                        None)
  --refSequence REFSEQUENCE
                        name of reference sequence within reference genome
                        (all sequences if empty) (default: None)
  --refTargets REFTARGETS
                        bed file coordinates of intervals in the reference
                        genome to export (default: None)
  --start START         coordinate within reference genome (or sequence if
                        specified) to start at (default: None)
  --length LENGTH       length of the reference genome (or sequence if
                        specified) to convert. If set to 0, the entire thing
                        is converted (default: None)
  --rootGenome ROOTGENOME
                        name of root genome (none if empty) (default: None)
  --targetGenomes TARGETGENOMES
                        comma-separated (no spaces) list of target genomes
                        (others are excluded) (vist all if empty) (default:
                        None)
  --maxRefGap MAXREFGAP
                        maximum gap length in reference (default: None)
  --noDupes             ignore paralogy edges (default: False)
  --noAncestors         don't write ancestral sequences. IMPORTANT: Must be
                        used in conjunction with --refGenome to set a non-
                        ancestral genome as the reference because the default
                        reference is the root. (default: False)
  --onlySequenceNames   use sequence names for output names. By default, the
                        UCSC convention of Genome.Sequence is used (default:
                        False)
